# Tooling images for Smee


<img src="https://raw.githubusercontent.com/hauke-cloud/.github/main/resources/img/organisation-logo-small.png" alt="hauke.cloud logo" width="109" height="123" align="right">


This library is a collection of container images which provide easy to use tooling environment, used by Smee.

What you can do with it:
- Use tools for specific development environment (opentofu/terraform, helm, etc.)
- Enjoy pre-configured terminal with zsh, spaceship, etc.
- Use a huge amount of pre-configured alias and functions
